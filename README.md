# Weather Dashboard App
## Purpose
To allow a user to search a city and state and pull weather info for the current date and a 5 day forecast
## Frameworks implemented
- Bootstrap
- Moment.js

## Server Side APIs fetched
- OpenWeather API
- Geonames API

## Built with
- HTML
- CSS
- Javascript
